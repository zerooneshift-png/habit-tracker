# habit-tracker

Daily habit tracking app with streaks, reminders, and progress charts.

## Features

- Daily tracking
- Streak counter
- Reminders
- Progress charts

## Tech Stack
- HTML5
- CSS3 (Flexbox, Grid, Animations)
- JavaScript (ES6+)

## Live Demo
Open `index.html` in any modern browser to view the website.

## File Structure
```
habit-tracker/
├── index.html      # Main HTML file
├── style.css       # Styles and layout
├── script.js       # JavaScript functionality
└── README.md       # Project documentation
```

## Customization
1. Edit `index.html` to update text content and structure
2. Modify `style.css` to change colors, fonts, and layout
3. Update `script.js` for interactive features
